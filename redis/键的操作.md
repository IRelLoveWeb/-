## 获取符合条件的列名 
* ``` KEYS pattern```
* pattern是正则匹配 ？单个字符, * 0或多个字符
```  
    KEYS ba?
    KEYS ba*
    KEYS ba[a-z]
    KEYS ba\x
```

## 判断键是否存在 (存在返回1, 不存在返回0)
``` EXISTS key```

## 删除键 (可以删除多个键, 返回删除键的个数)
* ``` DEL k1 k2```
* 单条命令删除多个键(非交互环境中执行) ``` redis-cli DEL `redis-cli KEYS "bar*"` ```

## 获取键类型
``` TYPE key```
