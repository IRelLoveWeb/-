### 1.事物 支持操作 insert, update, delete
```
START TRANSACTION;
update tablename set col1 = xx;
insert into tablename values (xx, xx);
delete from tablename where col1 = xx;
COMMIT; / ROLLBACK;
```

### 2.一般的insert, update, delete 操作会进行隐含提交,即自动commit

### 3.ROLLBACK 回退

### 4.保留点  部分回退
```
SAVEPOINT delete1;
ROLLBACK TO delete1;
```

### 5.设置sql语句不自动提交更改
```
SET autocommit = 0;
```
