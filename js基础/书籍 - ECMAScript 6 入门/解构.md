## 解构
> 从数组和对象中提取值, 对变量进行赋值

## 数组解构(按顺序依次解构)
* 有iterator接口的数据解构, 都可以用数组方式来解构
* 解构不完全, 解构值为undefined
  > `let [x] = []; // x = undefined`
* 解构默认值
```
  let [x = 2] = []; // x = 2 
  let [x = 2] = [undefined]; // x = 2
  let [x = 2] = [null]; // x = null
  let [x, ...rest] = [1, 2, 3] // x = 1, rest = [2, 3]
```

## 对象解构
```
  let { x } = { x: 1 }    // x = 1
  let { x: y } = { x: 1 } // y = 1
  let { x = 2 } = {}      // x = 2
  let { x: z = 3 } = {}   // z = 3
```

## 扩展运算符(...)
```
  [...[1, 2, 3]]
  [...'123']
  [...new Set([1, 2, 3])]
  [...new Map([[1, 2], [3, 4]])]
```
